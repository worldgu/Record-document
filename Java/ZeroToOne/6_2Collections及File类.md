4. 流的分类

   1. 按流向进行划分

      1. 输出流
         - OutputStream和Writer作为分类
      2. 输入流
         - InputStream和Reader作为基类

      *输入输出流是相对与计算机内存来说的，而不是相对于源和目标*

   2. 按照处理数据单元划分

      1. 字节流
         1. 字节输入流InputStream基类
         2. 字节输出流OutputStream基类
      2. 字符流
         1. 字符输入流Reader基类
         2. 字符输出流Writer基类

      *字节流是8位通过字节流，字符流是16为Unicode字符流*

   - 字节是物理划分的、字符是逻辑划分的
   - 文字使用字符流，  图片之类的信息使用的是字节流
