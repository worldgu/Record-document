### 长连接与短链接

- 长连接
  - 游戏服务器
  - 可以主动推送数据
  - 传输二进制数据
  - 协议自己攒
  - 占用资源相对较多
- 短链接
  - Web服务器
  - 被迫营业
  - 传输文本数据
  - Http、Html
  - 占用资源相对较少
